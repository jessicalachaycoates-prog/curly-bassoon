# Data Engineering Learning Labs

This repository contains various projects and labs designed to enhance learning in data engineering. Below is an overview of the project structure:

## Project Structure

```
curly-bassoon/
├── lab_1/
│   ├── data/
│   ├── notebooks/
│   └── scripts/
├── lab_2/
│   ├── data/
│   ├── notebooks/
│   └── scripts/
└── README.md
```

- **lab_1**: Contains materials and code for the first lab, including data files, Jupyter notebooks, and scripts.
- **lab_2**: The same structure is followed for the second lab, allowing for organized access and easy navigation.
- **README.md**: This file provides an overview of the project.

## Getting Started

1. Clone the repository: `git clone https://github.com/jessicalachaycoates-prog/curly-bassoon.git`
2. Navigate to the desired lab folder to access materials.

Happy learning!
